# ZTE E8820S ImmortalWrt

This fork has immortalwrt official image zte e8820s but patched for hc5962 layout

This fork also added:

- Tailscale + kmod-wireguard
- Argon Theme
- DDNS
- USB WAN
- ZRAM
- OpenAppFilter
- iperf3
- UPnP
- SQM

## Default login

```
URL = 192.168.1.1
user = root
no password, just hit enter

Wi-Fi ssid = ImmortalWrt
Password = <no>
```
